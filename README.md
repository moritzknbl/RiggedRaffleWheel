# Usage

1. Enter Names: Type participants into the text box (one per line).
2. Spin: Click the "Spin Now" button to start the wheel.
3. Rigging (Optional): Click the Settings (Gear) icon in the top right corner. Enter the exact name of a participant to force the wheel to land on them.
